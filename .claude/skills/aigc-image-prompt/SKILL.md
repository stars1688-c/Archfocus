# AI 配图生成提示词 Skill

根据小红书笔记文案（标题 + 正文 + 标签），自动生成适配的 AI 绘图提示词。Agent 读取本 Skill 的规则后，对文案进行分析并输出结构化提示词。

## 使用方式

Agent 节点调用本 Skill 时的流程：

1. **输入**：笔记标题 + 正文内容 + 标签（可选）+ 账号人设风格（可选）
2. **处理**：分析内容类型 → 确定构图规则 → 应用风格预设 → 组合提示词
3. **输出**：完整 AI 绘图提示词（Image 2.0 / SD 兼容格式）

## 内容类型识别

根据文案内容自动判断配图类型：

| 类型 | 关键词 | 适用场景 |
|------|--------|----------|
| `cover` | 推荐、TOP、盘点、排行、汇总 | 清单封面、合集类笔记 |
| `tutorial` | 教程、步骤、如何、方法、操作 | 教学类、教程类笔记 |
| `data` | 数据、统计、分析、报告、对比 | 数据复盘、效果对比 |
| `compare` | 对比、vs、区别、前后 | 产品对比、效果对比 |
| `list` | 清单、推荐、盘点、排行 | 好物推荐、清单类 |
| `lifestyle` | 日常、分享、开箱、探店 | 生活记录、开箱测评（默认） |

## 构图规则

### cover（封面型）
- 竖版 3:4，标题区占画面上部 30%
- 要点区居中排列 3-6 个核心要点
- 署名区底部预留 15% 空间
- 视觉重心突出标题主题
- 色板 3-4 个主色，标题用主色突出

### tutorial（教程型）
- 竖版 3:4，自上而下垂直排列步骤
- 步骤间用箭头/连线/渐变背景形成视觉引导
- 每个步骤编号清晰（1-2-3）
- 步骤说明简洁，每步不超过 20 字
- 统一的插画/摄影风格

### data（数据型）
- 竖版 3:4 仪表盘布局
- 核心数据用大号字体在顶部 30% 区域
- 中间展示迷你图表或进度条
- 底部补充说明和对比数据
- 数据可视化优先，避免文字堆砌

### compare（对比型）
- 竖版 3:4 左右或上下分割
- 对比项占据相等空间
- 中间用分割线或色块区分
- 重点差异用箭头/圆圈标注
- 对比色但和谐统一

### list（清单型）
- 竖版 3:4 清单式布局
- 编号/图标/色块标记每个条目
- 条目垂直排列，间距适中
- 前 3 名或推荐项视觉突出
- 色彩搭配有节奏感

### lifestyle（生活型）
- 竖版 3:4 自然构图
- 主体居中，适当留白
- 场景化布局，有氛围感
- 日常真实感优先

## 通用视觉约束

无论何种类型，必须遵守以下规则：

1. **比例**：强制 3:4 竖版（1024×1536）
2. **安全区域**：四周预留 10-15% 安全距离，关键信息集中在中央 70% 区域
3. **字体**：中文为主，最小字号不低于整体高度 3%，避免小字密集
4. **色板**：严格控制 3-4 个主色，避免彩虹色或过于鲜艳的组合
5. **禁止**：不添加平台水印，不伪造小红书界面截图
6. **风格统一**：同一图片内保持统一的插画或摄影风格
7. **质量**：光线自然真实，避免过度磨皮或失真，细节清晰

## 风格预设

### ins 风
- 适用：时尚、美妆、生活
- 特点：高对比度，轻微复古滤镜感，自然光线
- 配色：`#D4A574` 主 / `#8B7355` 辅 / `#F5E6D3` 强调
- 参数：`style=photorealistic, contrast=high, lighting=natural`

### 极简风
- 适用：知识、职场、科技
- 特点：大量留白，几何构图，无多余装饰
- 配色：`#333333` 主 / `#666666` 辅 / `#E8E8E8` 强调
- 参数：`style=minimal, contrast=medium, lighting=bright_even`

### 治愈温暖风
- 适用：家居、亲子、情感、美食
- 特点：柔和暖色调，温馨生活感，柔光
- 配色：`#F4A460` 主 / `#CD853F` 辅 / `#FAEBD7` 强调
- 参数：`style=soft, contrast=low_medium, temperature=warm`

### 复古风
- 适用：港风、复古时尚、胶片感
- 特点：胶片质感，怀旧色调偏黄偏红，故事性构图
- 配色：`#8B4513` 主 / `#D2691E` 辅 / `#F0E68C` 强调
- 参数：`style=vintage, contrast=medium, filter=filmgrain`

### 清新自然风
- 适用：植物、户外、健康、日系
- 特点：明亮通透，淡雅清新色调
- 配色：`#90EE90` 主 / `#98FB98` 辅 / `#F0FFF0` 强调
- 参数：`style=fresh_natural, contrast=medium, lighting=bright_natural`

```

```

## 输出格式

Agent 输出以下结构给工作流：

```json
{
  "prompt": "完整的 AI 绘图提示词文本",
  "content_type": "cover|tutorial|data|compare|list|lifestyle",
  "style": "ins|minimal|warm|vintage|fresh",
  "params": {
    "size": "3:4",
    "resolution": "2k",
    "quality": "high"
  }
}
```

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| size | 3:4 | 强制竖版，最适小红书 |
| resolution | 2k | 默认 2K，data 类型自动 4K |
| quality | high | 高质量 |
| style_boost | - | tutorial 类型追加 `clear_steps` |
