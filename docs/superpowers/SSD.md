# ArchFocus 创作页面规格说明书 (SSD)

> **版本：** v1.0
> **日期：** 2026-06-07
> **基于：** prototype/index.html + PRD.md
> **模型策略：** 文案模型固定为 MiniMax，不提供切换选项

---

## 1. 功能概述

创作页面是四步向导流程：选题 → 文案 → 配图 → 发布。用户完成每一步后可进入下一步，也可返回上一步修改。

---

## 2. Step 1 - 选题

### 2.1 发布账号显示

**UI 形式：** 文本显示（只读），非下拉框

```
┌─────────────────────────────────────────────────────┐
│ 发布账号                                             │
│ ┌─────────────────────────────────────────────────┐ │
│ │ 美妆小魚 🎀    小红书 ID: meizhuang_xiaoyu     │ │
│ └─────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────┘
```

**实现：**
- 从全局状态获取当前选中账号
- 显示：账号名称 + 小红书 ID（如果有）
- 纯展示，不可编辑

### 2.2 数据源多选

**UI 形式：** Toggle 按钮组，可多选

| 选项 | 图标 | 说明 |
|------|------|------|
| 联网搜索 | 🌐 | 调用 MiniMax 联网搜索 |
| 小红书行业热点词 | 🔥 | 使用热点词数据 |

**状态：** 默认联网搜索选中

**实现逻辑：**
```typescript
interface DataSource {
  webSearch: boolean    // 联网搜索
  hotWords: boolean   // 热点词
}
```

### 2.3 主题输入方式

**两种方式：**
1. **AI 推荐** - 点击按钮调用选题生成 API
2. **手动输入** - 直接输入主题名称

### 2.4 推荐主题展示

**UI：** 卡片列表，点击选中

```
推荐主题（点击选择）
┌─────────────────────────────────────────────────────┐
│ 🌞 夏日持妆大法！油皮也能 hold 住一整天              │
│ 基于热点"夏季化妆" + 你的人设"美妆"                │
└─────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────┐
│ 💸 百元以内学生党彩妆红黑榜                         │
│ 基于热点"平价彩妆" + 你的人设"学生党"             │
```

**卡片内容：**
- 标题（emoji + 主题）
- 推荐理由（基于什么数据源 + 人设）

**Input placeholder：** "手动输入笔记主题..."

## 10. API 接口

### 10.1 选题生成

**请求：** `POST /api/workflow`
```typescript
{
  account: Account
  topic?: string           // 手动输入的主题
  searchEnabled: boolean   // 联网搜索
  hotWordsEnabled: boolean  // 热点词
}
```

**响应：**
```typescript
{
  success: boolean
  data: {
    topics: Array<{
      title: string
      reason: string  // 推荐理由
    }>
  }
}
```

### 10.2 文案生成

**请求：** `POST /api/workflow`
```typescript
{
  account: Account
  selectedTopic: string
  searchEnabled: boolean
  userRequirements?: string  // 写作要求
}
```

### 10.3 配图生成

**请求：** `POST /api/workflow/image`
```typescript
{
  content: string           // 完整文案
  title: string
  imageType: 'ai_prompt' | 'html_screenshot'
  imageModel?: 'gpt-image-2' | 'doubao-seedream-5-0-lite'  // 仅 ai_prompt
  imageCount?: number       // 仅 ai_prompt, 1-9
  htmlStyle?: 'magazine' | 'cream' | 'forest' | 'minimal' | 'dopamine'  // 仅 html_screenshot
}
```

## 12. 验收标准

### Step 1 - 选题
- [ ] 发布账号以文本形式显示（非下拉框）
- [ ] 数据源支持多选（联网搜索 + 热点词）
- [ ] 推荐主题显示理由
- [ ] 主题可点击选中

### Step 2 - 文案
- [ ] 无文案模型选择（固定 MiniMax）
- [ ] 有写作要求输入框
- [ ] 文案可内联编辑（contenteditable）
- [ ] 敏感词徽章显示正确
- [ ] 重新生成弹窗可输入反馈

### Step 3 - 配图
- [ ] AI+提示词：数量 1-9 可选
- [ ] AI+提示词：模型可选 Image 2.0 / Seedream
- [ ] HTML→截图：5种风格可选
- [ ] HTML→截图：自动计算分页数量
- [ ] 图片预览网格显示

### Step 4 - 发布
- [ ] 标题可复制
- [ ] 内容可复制
- [ ] 配图列表显示，每项可复制
- [ ] 定时发布弹窗完整
- [ ] 邮件提醒开关可用
